## 🐛 Common First Steps

1. **If npm install fails**
   ```bash
   rm -rf node_modules
   npm install
   ```

2. **If port 5173 is in use**
   ```bash
   npm run dev -- --port 3000
   ```

3. **If styles don't load**
   - Clear browser cache (Ctrl+Shift+Delete)
   - Restart dev server

4. **If animations skip**
   - Check GPU acceleration (DevTools > Settings)
   - Browser compatibility
